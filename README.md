## VILOOK TECH App Releases Downloader

This is a professional GitHub Pages site for VILOOK TECH CO., LTD to display and allow downloading of app releases from the GitHub repository, grouped by app.

### Features
- Displays company information prominently.
- Fetches releases from the specified GitHub repo.
- Skips releases containing "check_for_update_info" in the name.
- Groups releases by app prefix (e.g., "deep_logic" from "deep_logic_setup_0.1.0").
- Search functionality to filter apps.
- Shows only the latest version by default per app, with a toggle to show all versions.
- Displays release notes and download links for each version.

### Setup

1. Ensure `script.js` has the correct `owner` and `repo` (currently set to 'vilook-tech' and 'resource').
2. Push these files to your GitHub repository.
3. Go to your repository settings > Pages, and set the source to "Deploy from a branch" and select the main branch.
4. The site will be available at `https://vilook-tech.github.io/resource/`
